# Ecommerce-AI-voice-text-Assistant

A bilingual voice-driven conversational assistant designed to make e-commerce more accessible for users in Pakistan. The system supports voice and text commands in English, Roman Urdu, and Urdu, allowing users to search for products, manage their cart, and confirm orders through a simple conversational interface.

This project was developed as a Human-Computer Interaction (HCI) prototype by students of the Department of Computer Engineering, Information Technology University (ITU), Lahore.

## Features

- 🎤 Voice-based product search
- 💬 Text input support
- 🌐 Multilingual interaction:
  - English
  - Roman Urdu
  - Urdu
  - Code-switched commands
- 🛒 Add/remove products from cart
- ✅ Order confirmation through conversation
- 🖥️ User-friendly and accessible interface
- ⚡ MERN stack implementation

## Tech Stack

### Frontend

### Backend
- Node.js

### Database
- MongoDB

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/Ecommerce-AI-voice-text-Assistant.git
cd Ecommerce-AI-voice-text-Assistant
```

## Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

Open another terminal:

```bash
cd server
npm install
```

## Run the Project

Run frontend and backend in separate terminals.

### Terminal 1 — Frontend

```bash
cd client
npm run dev
```

### Terminal 2 — Backend

```bash
cd server
npm start
```

Then open the frontend in your browser.

## Accessibility Focus

This project aims to improve e-commerce usability for:

- Urdu-speaking users
- Older adults
- Users with limited typing ability
- Low-literacy users
- People who prefer voice interaction
