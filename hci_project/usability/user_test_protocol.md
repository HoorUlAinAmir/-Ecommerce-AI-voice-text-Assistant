# CP3 Usability Evaluation Protocol

Use this for the required 3-5 user evaluation before final PDF submission.

## Participants

Recruit 3-5 users. Try to include:
- at least one Urdu-dominant user,
- at least one user who prefers Roman Urdu,
- at least one digitally fluent user for comparison.

Planned participant names:
- Hassan
- Mateen
- Saad
- Maryam
- Rameeza

## Tasks

Ask each participant to complete these tasks using voice or typed fallback:

1. Search for a blue kurta.
2. Search for red shoes under PKR 5000.
3. Add one product to the cart.
4. View the cart.
5. Confirm or cancel the order.

## Observation Checklist

Record whether the user:
- mixed English and Urdu naturally,
- understood the assistant's feedback,
- noticed the detected intent/slots,
- recovered after a misunderstood command,
- trusted the final order confirmation step.
